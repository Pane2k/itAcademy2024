Тэг <code>
# This program prints Hello, world!

print('Hello, world!')
Тэг <tt>
Тэг для моноширинного вывода текста на html странице. Можно использовать точно так же как <code>.
Тэг <kbd>

Используется для обозначения текста, который набирается на клавиатуре или для названия клавиш, например для копирования нужно использовать сочинение клавишь <ctrl>+<c> а для вставки <ctrl>+<v>.

Тэг <var>

Используется для выделения переменных в программах, например - content содержит в себе данные об объекте контента.

Тэги <sup> и <sub>

Используется для верхнего и нижнего индекса, к примеру a1 + a2 + a3 + ... + an = xn

Тэги <blockquote> и <q>
Тэг <blockquote> предназначен для выделения длинных цитат внутри документа, так же добавляет отступы по две стороны от текста. Специально длинный текст чтобы увидить как оно работает.

А это тэг <q>, он автоматически выделяет текст в ковычки, используется для цитат.
Тэги <ins> и <del>
Если надо зачеркнуть какой либо бред текст, но оставить его то используем тэг <del>, а если надо подчеркнуть что либо в тексте то используем тэг <ins>.
Тэг <address>
Тэг <address> используется для хранения информации об авторе. Используется для поисковых роботов.
Тэги <nobr> и <wbr>
Тэг <nobr> уведомляет браузер отображать текст без переносов строк на каком либо тексте. Если текст достаточно длинный то он будет выходить за границы страницы, этото тэг убирает автоматический перенос строк

А тэг <wbr> позволяет указать где необходимо сделать пере­нос какого либо слова, попробуйте сделать браузер чуть не­много уже. Можно еще применить символ мягкого переноса &shy;
